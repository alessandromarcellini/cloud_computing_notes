DOCs middlewares are a type of middleware **not so inexpensive** that enables client server blocking communication between entities in a seamless way, providing **more dynamicity than RPCs** and direct access to **Objects themselves** because the client is not really tied to the client, there's no way the client can store the identity of the server.
Moreover there could be **different servers** that implement the same interface and it is a **middleware's role to choose** the best fitting for the client.
