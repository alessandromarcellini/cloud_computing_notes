MOMs are middlewares that are based on **message passing protocols**, enabling different entities to communicate with each other easily and adding functionalities and features to the communication (such as bufferization, asynchronous sending, delaying the communication, making the communication asynchronous and so on).

Its common and most important features are:
- Enable **asychronous** communication between clients;
- Usually enable broadcast or **multicast** to different entities;
- **Messages** are typically **persistent**.
